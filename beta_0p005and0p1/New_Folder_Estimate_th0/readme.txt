
注意：fn_Sh_Entropy: 计算信息熵用的是log2


程序运行先后：main_S1.m --> main_S2.m --> main_S3.m
                      main_S1.m --> main_S1_Fig.m
                      main_S3.m --> main_S3_Fig.m
                      main_S_Fermi_surface.m


main_S1.m : 计算测演化时间间隔较小 (等价测量强度强) 的香农熵和Fisher信息，其中平均次数为1000。
                    并记录下每个演化路径中每次测量的结果和最终的矩阵 A 和 B ，
                    其中A是每次测量后是直接归一化，B是每次测量后是QR归一化，保证归一化后还是幺正的。
注：数值计算过程中会误差积累，导致归一化出现误差，主要体现在计算被测概率vP的求和不等于1。
      程序中设置了当误差大于5*1e-8时会输出误差值，目前出现的最大误差大于1.e-8，但未超过设定的阈值，影响可以忽略。超过阈值也没关系，只要不是太大就行，目前没有更好避免或缩小这个误差的方法。


main_S2.m : 参数估计。给定好几个可能的参数值th，通过计算相应的似然概率，选出概率最大的参数th_old0。

main_S3.m : 参数估计。用最速下降法找最大似然概率所对应的参数th，为了避免陷入局部最优。
                    用main_S2.m中得到th_old0作为初始th。
                    其中，fn_find_max是通过求导的方式实现最速下降法，
                    fn_find_max_old1是通过求周围的几个点进行下降的方法。
注：在用下降法的时候，有可能遇到的一个问题是，如果每次移动的步长lr取太小，则可能会走得很慢花很长时间才到最优点；如果步长取太大，容易陷入在最优点附近反复跳跃的死循环，而无法达到最优点。目前的步长花费的稍微有点慢，但是总体还行，能接受。


main_S_Fermi_surface.m : 计算并绘制每次测量前每个格点上被测量的概率和每个格点上有粒子占据得概率的图。

main_S1_Fig.m : 绘制统计平均后香农熵，Fisher信息和纠缠熵随时间演化的图。

main_S3_Fig.m : 计算并绘制每条演化路径的结果所估计的待测参数，以及它们的统计平均分布图。


main_W系列也和上面是一样的，只是演化时间不同，S系列选择beta=0.005，W系列选择beta=0.1。
